Orchestra-register by timhunt

A simple web application to manage attendance at orchestra rehearsals. It could probably also be adapted for other similar groups.

Orchestra Register comes with documentation build in. You can browse that documentation online.

Author

Tim Hunt (T.J.Hunt@open.ac.uk)

Download

You can download the latest stable release (Version 2.0) in either zip or tar formats.

Alternatively, you can download the latest development version (Version 3.0 beta) for testing in either zip or tar formats.

You can also clone the latest development version with Git by running: 

$ git clone git://github.com/timhunt/Orchestra-register
get the source code on GitHub : timhunt/Orchestra-register
